VulScan说明

使用说明

遵循高内聚、低耦合、轻量级 vulscan poc管理工具，基于pocsuite,web界面基于 django
集成Pocsuite-dev、支持扩展模块、扩展POC
Template.py 是 pocsuite规范 模版,里面有使用参考手册,可以单独使用 pocsuite 进行交叉扫描
先要安装python依赖库,pip install -r requirements.txt,然后切换到目录运行即可 cd ~ && start.bat

API 说明

接口参数：action=[index|调用一键扫描,jindu|获取扫描进度,report|获取扫描报告],url=test.com|扫描的url,token=api|扫描用户名
一键扫描接口：http://127.0.0.1:8000/api?action=index&url=127.0.0.1&token=api
扫描进度接口：http://127.0.0.1:8000/api?action=jindu&url=127.0.0.1&token=api
json接口报告：http://127.0.0.1:8000/api?action=report&url=127.0.0.1&token=api

主要功能

一键扫描

一键扫描是根据本地存在的poc，一一加载做扫描，一键扫描填写的url根据实际情况填写（域名，路径），开始扫描后按钮会显示实时扫描进度，扫描完成后会出现查看报告按钮！

POC 插件

POC插件主要分为两类，分别为主机类和WEB类型的漏洞，可以对单一一个漏洞进行扫描，对于自定义的插件，复制进 vul目录下，重启即可看到和使用poc插件了！

端口扫描

端口扫描使用默认快速方式扫描常见端口和服务，判断不可信或存在风险的端口进行提示！

基线检查

基线检查目前只支持linux衍生系统，支持在线检查和离线检查，输入ip，用户，密码即可自行检查系统及三方软件的安全配置，操作系统等漏洞，主要用于白盒检查。实用为主！

网址导航

支持通过admin后台配置导航分类及导航内容，记录常用的url便于收藏和学习！

扩展模块
